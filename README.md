# Exploratory Data Analysis on MPG Dataset

## Overview
This project performs Exploratory Data Analysis (EDA) on the MPG dataset to understand factors affecting fuel efficiency.

## 🛠 Tools
- Python
- Pandas
- Seaborn
- Matplotlib

## Key Insights
- Weight negatively impacts MPG
- Newer cars tend to be more fuel efficient
